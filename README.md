# ikkyu-3
ikkyu-3専用Repository

## test_01
jsのスキルチェックです。

【追記】gulp導入しました(test_01ディレクトリ内で実行してください)
1. node.jsをinstall(homebrewの場合)  
`brew install node`

2. gulpをinstall  
`npm install gulp-cli -g`
`npm install gulp -D`

3. package.jsonからnode_modulesをinstall  
`npm install`

### 使い方
1. test_01ディレクトリに移動  
`cd test_01`

2. コマンド実行  
  2-1. 問題用のmain.jsを書き出す場合  
  `gulp`  
  2-2. 解答用のmain.jsを書き出す場合  
  `gulp answer`
